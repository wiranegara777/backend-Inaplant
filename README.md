# backend-Inaplant
Backend RSET API dari Aplikasi Controll Monitoring Perkebunan

### how to install backend

 - Install PHP v7
 - Clone the repo
 - Move repo to htdocs folder
 - turn on xampp (APACHE and mysql)
 - import plant.sql to database mysql

### enter this command

```sh
$ composer update
$ php artisan serve
```
